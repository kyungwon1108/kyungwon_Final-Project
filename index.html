Predicting hypertension using health insurance data

Kyungwon Lee 1
1Division of Digital Healthcare, Yonsei University
Code

Introduction

고혈압은 조기 발견과 예방이 중요한 만성질환으로, 혈압 측정 없이도 건강검진 데이터만으로 고혈압을 예측할 수 있다면 여러 가지 중요한 이점을 제공합니다. 첫째, 추가적인 검사를 받지 않아도 되므로 접근성이 향상되고, 시간과 비용을 절감할 수 있습니다. 둘째, 비침습적인 예측 방법은 환자의 불편함을 줄여주며, 정기적인 건강검진 데이터를 통해 고혈압 위험을 지속적으로 모니터링할 수 있습니다. 본 보고서에서는 이러한 예측 프로그램의 개발 과정과 그 결과에 대해 자세히 살펴보겠습니다.

Method

1. 데이터 전처리 및 모델 학습: 건강검진 데이터를 전처리하고, Logistic Regression과 XGBoost 모델을 사용하여 고혈압 예측 모델을 학습시킵니다.
2. 모델 평가: 각 모델의 예측 성능을 평가하고, 혼동 행렬과 ROC 곡선을 시각화하여 성능을 분석합니다.
3. 데이터 분포 시각화: 라벨 분포와 혈압 분포를 시각화하여 데이터의 특징을 파악합니다.

Body

1. 본 프로젝트에서는 건강검진 데이터를 사용하여 고혈압 예측 모델을 개발하였으며, 데이터 전처리 과정에서는 고혈압 라벨링, 결측치 처리, 범주형 변수 인코딩, 언더샘플링을 통한 데이터 균형 조정, 데이터 표준화를 수행하였습니다. 이를 통해 학습 데이터와 테스트 데이터를 준비하여 모델의 성능을 향상시켰습니다.

2. 본 프로젝트에서는 두 가지 인공지능 모델을 사용하여 고혈압을 예측하였습니다. 첫 번째 모델은 Logistic Regression으로, 이는 선형 회귀 모델의 확장 버전으로 분류 문제에 적합한 모델입니다. 두 번째 모델은 XGBoost로, 이는 부스팅 기법을 사용하여 여러 약한 학습기를 결합하여 강한 학습기를 만드는 앙상블 기법 중 하나입니다. 각 모델은 학습 데이터로 학습시킨 후, 테스트 데이터로 성능을 평가하였습니다.

3. 본 프로젝트에서의 실험 결과는 Logistic Regression과 XGBoost 모델의 고혈압 예측 성능을 나타냅니다. 각 모델의 정밀도, 재현율, F1-점수, 지원 수치를 통해 두 모델의 성능을 평가할 수 있습니다.
Logistic Regression:
클래스 0 (정상): 정밀도는 0.66, 재현율은 0.64, F1-점수는 0.65로 나타났습니다. 이는 모델이 정상인 사람들을 비교적 정확하게 예측할 수 있음을 나타냅니다.
클래스 1 (고혈압): 정밀도는 0.65, 재현율은 0.66, F1-점수는 0.66으로 나타났습니다. 이는 모델이 고혈압인 사람들을 비교적 잘 찾아낼 수 있음을 의미합니다.
XGBoost:
클래스 0 (정상): 정밀도는 0.68, 재현율은 0.61, F1-점수는 0.64로 나타났습니다. 이는 모델이 정상인 사람들을 예측하는 데 있어 다소 낮은 재현율을 보이지만, 정밀도는 높습니다.
클래스 1 (고혈압): 정밀도는 0.64, 재현율은 0.71, F1-점수는 0.67로 나타났습니다. 이는 모델이 고혈압인 사람들을 비교적 잘 예측하고 있음을 나타냅니다.
두 모델의 전체적인 성능을 비교하면, XGBoost 모델이 약간 더 높은 정밀도와 재현율을 보이며, 고혈압 예측에 있어 더 나은 성능을 보입니다. 특히, XGBoost 모델은 클래스 1 (고혈압)에 대해 더 높은 재현율을 보여, 고혈압 환자를 더 잘 찾아내는 경향이 있습니다.
Logistic Regression 모델은 클래스 0 (정상)과 클래스 1 (고혈압) 모두에서 균형 잡힌 성능을 보였으며, XGBoost 모델은 클래스 1 (고혈압)에 대해 더 나은 성능을 보였습니다.

Results

본 연구는 건강검진 데이터를 활용하여 혈압을 예측하는 모델을 개발하고 평가하였습니다. 데이터 전처리 과정을 거쳐 Logistic Regression과 XGBoost 모델을 사용하여 예측 성능을 비교한 결과, XGBoost 모델이 더 높은 정밀도와 재현율을 보였습니다. 이를 통해 고혈압 조기 발견과 예방에 있어 데이터 기반 접근 방식의 유효성을 확인하였습니다.
